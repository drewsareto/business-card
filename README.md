# Digital Business Card

## About the Project

This is a simple HTML-only Digital Business Card created as part of the Power Learn Project (PLP) Web Development course.

The webpage introduces me, my interests, experience, and provides links to my social media profiles.

## Features

* Personal introduction
* Profile image
* List of interests and hobbies
* Experience section
* GitHub, Instagram, and TikTok links
